
# SMS Communication Device
## R&D Internship Project

## Overview
A Raspberry Pi-based SMS communication device developed during my R&D 
Internship at IoT Technology, Incorporated. The device integrates a 
SIM808 GSM module for SMS operations, and is housed in a custom 
3D-printed enclosure designed to fit the hardware layout for a clean, 
deployable form factor.

## Features
- Bulk SMS sending
- Person-to-person messaging
- Group sending support
- Custom 3D-printed enclosure for deployment-ready packaging

## Hardware
- Raspberry Pi
- SIM808 GSM Module
- Custom 3D-printed casing
