# SMART Voltage Leak Detection System

## Overview
A prototype system designed to simulate smart voltage leak 
detection for household electrical systems. The system monitors 
voltage levels and automatically responds when an abnormal spike 
is detected — shutting off power, checking for leaks, calculating 
excess voltage, and only restoring power once voltage returns 
to normal levels.

## Tech Stack
- Arduino
- Voltage Sensor
- LCD Display
- Relay Switch
- C++

## How It Works
1. System continuously monitors household voltage levels
2. When a sudden spike beyond normal threshold is detected
3. Relay switch automatically cuts off power
4. System checks and calculates the excess voltage
5. Power is only restored once voltage returns to normal
