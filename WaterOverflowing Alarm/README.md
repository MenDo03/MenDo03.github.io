# Water Overflowing Alarm System

## Overview
A prototype alarm system designed to monitor water levels in 
containers and alert users before overflow occurs. Built as a 
practical solution for everyday water management — useful for 
water tanks, containers, and similar applications.

## My Role
Group project with 4 members. I handled:
- Hardware assembly and wiring
- Arduino programming and system logic

## Tech Stack
- Arduino
- Water Level Sensor
- LEDs (indicator lights)
- Buzzer
- C++

## How It Works
The system uses a water level sensor to continuously monitor 
water levels with three indicator states:

| LED Status | Water Level |
|------------|-------------|
| 🟢 Green | Normal Level |
| 🟡 Yellow | High Level |
| 🔴 Red + Buzzer | Overflow Warning |

When water reaches a critical level the buzzer activates to 
alert the user that immediate attention is needed.

## Testing
Tested on water containers requiring filling to validate 
sensor accuracy and alarm response time.
